# Kin Thai Restaurant 🍜

Modern Thai dining in the heart of Nantwich, England.

## About

Kin Thai Restaurant is a family-run Thai restaurant bringing authentic Thai flavours with a modern twist to Nantwich, Cheshire. Built on the legacy of Aroi-Thai, we offer a welcoming dining experience for everyone.

## Website

This is a static website built with HTML, CSS, and JavaScript, hosted on GitHub Pages.

### Features

- Responsive design for all devices
- Bilingual support (English / Thai)
- Interactive menu with allergy information
- Online booking integration (ResDiary)
- Delivery & Click & Collect information

## Getting Started

### GitHub Pages

1. Go to repository **Settings** → **Pages**
2. Set source to **main** branch, folder `/ (root)`
3. Your site will be live at `https://<username>.github.io/kin-thai-restaurant/`

### Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```
